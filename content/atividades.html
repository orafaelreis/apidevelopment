Atividade #2

Construir uma API sobre as grandes personalidades do nosso país, como Machado de Assis, Santos Dumont, Carmen Miranda, etc.
Entidades e atributos que podem estar presentes na API:

  • Personalidade (id, nome, data de nascimento, naturalidade)
  • Conhecimento (id, nome)
  • Premio (id, nome)
  • Feito (id, descrição)
  • Cidade (id, nome)
  • Estado (id, sigla)

O que deverá ser entregue no arquivo Zip:

  • Coleção dos endpoints (via Postman Collection)
  • Projeto NodeJs

Metodologia da Avaliação:
Funcional:
- Coleção dos enpoints entregues funcionam sem erro;
- Resultados não esperados são tratados.

Não-Funcional:
- Endpoints retornam o objeto corretamente;
- Verbos HTTP usados corretamente.
- Estrutura e Qualidade do código.

Lista de endpoints a serem desenvolvidos: 

GET /uf
✓ GET /ufs
✓ GET /uf/:slug

//endpoint para a entidade: Prêmio
GET /award/:id

DELETE /award/:id

//endpoint para a entidade: Conhecimento
GET /knowledge/:slug
POST /knowledge

DELETE /knowledge/:slug

//endpoint para a entidade: Personalidade
✓ GET /personalities (acrescentar nos resultados os feitos, áreas de conhecimento e premios da personalidade)
GET /personality/:slug
✓ POST /personality
PUT /personality/:slug
DELETE /personality/:slug

//endpoints adicionais
✓ GET uf/:slug/personalities
GET personality/:slug/awards
GET personality/:slug/achievements
