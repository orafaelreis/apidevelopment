Atividade #1

A Google disponibiliza acesso aos dados de várias de suas aplicações em forma de API.
Nessa disciplina vamos consumir a API do Youtube. Você deverá criar uma tela com um campo de busca que liste como resultado os videos procurados pelo usuário.

Atividade #2

Construir uma API sobre as grandes personalidades do nosso país, como Machado de Assis, Santos Dumont, Carmen Miranda, etc.
Entidades e atributos que podem estar presentes na API:

  • Personalidade (id, nome, data de nascimento, naturalidade)
  • Conhecimento (id, nome)
  • Premio (id, nome)
  • Feito (id, descrição)
  • Cidade (id, nome)
  • Estado (id, sigla)
