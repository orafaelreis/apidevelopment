Um pouco de História

1996: IFRAME
Invenção da Microsoft, o elemento IFRAME poderia incorporar uma outra página em sua página, que é uma maneira de trazer em outro servidor de uma forma simplista. Também poderia ser usado para trazer dados de forma assíncrona em uma página do mesmo servidor, semelhante ao que XMLHR tornaria possível mais tarde.

1999: XMLHttpRequest
XMLHttpRequest foi importante ao levar as pessoas a considerar a possibilidade de obtenção dos dados da web a partir de servidores de terceiros.

2004/2005: GMail, Google Maps
A Google explora todo o potencial do XMLHttpRequest e constrói o Gmail e o Google Maps.

Fev 2005: "AJAX"
Jesse James Garrett cunhou o termo "AJAX" para descrever a abordagem de buscar dados de forma assíncrona usando XMLHttpReq.

Jun 2005: Google Maps API
Google constrói a primeira versão da API do Google Maps

Dez 2005: "JSONP"
Devido às políticas de segurança (Same-Origin Policy) dos navegadores não é permitido o consumo/manipulação de dados entre páginas de diferentes domínios. Em Dez. de 2005, Bob Ippolitto descreve em seu blog uma tecnica ("hack") sobre a tag SCRIPT em que possível, de forma asíncrona trazer dados de outros servidores. Isso ficou conhecido como "JSONP".
Para a técnica funcionar, o servidor deve entender e processar chamadas JSONP. 

      <script type="text/javascript" src="http://api.anotherdomain.com/?q=somequery&callback=minhafuncao"></script>
